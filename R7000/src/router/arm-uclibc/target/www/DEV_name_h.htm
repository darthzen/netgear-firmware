Device Name Help

Device Name: device name identifies this router. Many router features provide this name to client devices connected for identification and display. If you change the device name, all following fields will be updated. You have the option to change or not change your ReadyShare storage access path.

Device Name Shown on the Router GUI:

  • Internet Setup Page: Account Name
  • LAN Setup Page: Device Name
  • ReadySHARE Media Server Page: Media Server Name
  • Router/AP/Extender mode, AP Mode Page: Device Name
  • Router/AP/Extender mode, Extender Mode Page: Device Name

Device Name only displayed on Client:

  • iTune Server Name
  • Time machine server name
  • ReadyShare printer / Bonjour print Server name
  • LLTD device name
  • UPNP device name
  • WPS device name
